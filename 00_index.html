    h tags are used to define headings in html
    h1 ---> is most important heading.
    h2 ---> is second important heading
    h3 ---> is third important heading
    h4 ---> is fourth important heading
    h5 ---> is fifth important heading
    h6 ---> is sixth important heading


    a (anchor) tags are used to define links in html
    Example:
     Google

    href attribute is used to define the link(define the path of the link)(link ko hold kerta hai)

    target attribute is used to define the target of the link(link kha per open hoga. Usi tab in ya fir new tab me)
    Example:
     Google
    _blank ---> open the link in new tab
    _self ---> open the link in same tab
    _parent ---> open the link in parent tab
    _top ---> open the link in top tab

    rel attribute is used to define the relationship of the link
    Example:
     Google
    noopener ---> prevent the new tab from accessing the window.opener property
    noreferrer ---> prevent the browser from sending the original referrer information to the linked page
